# Eduka Workshop - TDD 

This repository serves as a starter kit for doing simple TDD exercise

## Getting Started

### Dependencies

* NodeJS
* NPM
* Jest

### Installing

* Install dependencies
```
npm install
```

### Run tests

* To run test
```
npm test
```
